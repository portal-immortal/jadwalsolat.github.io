# 🌌 Portal Immortal

Portal Immortal adalah kumpulan mini web applications berbasis HTML & JavaScript yang dikembangkan sebagai:

- Eksperimen frontend
- Studi implementasi logika waktu & perhitungan
- Eksplorasi UI sederhana berbasis static hosting
- Playground Git & GitHub Pages

## 🕌 1. Jadwal Sholat App
Aplikasi jadwal sholat berbasis perhitungan waktu harian.

Fitur:
- Tampilan waktu dinamis
- Struktur modular HTML
- Siap dikembangkan ke API real-time

---

## ⏰ 2. IST ↔ WIB Converter
Konverter waktu sederhana untuk kebutuhan komunikasi lintas zona waktu.

Fitur:
- Perhitungan offset waktu
- Interface ringan
- Tanpa dependency eksternal
